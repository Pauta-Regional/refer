The Tool Bar

The Tool Bar is located below the program menu. It primary supports navigation. The first five buttons correspond to commands on the View Menu

The Back button navigates to the previous web page in the currently active child window. This item will be disabled if there isn't a previous web page.

The Forward button reverses the navigation of the Back button.

The Stop button stops the execution of a lengthy download.

The Refresh button refreshes the contents of the current page in the currently active child window.

The Home button goes to your Home page, which you can set on the Favorites Menu.

The Print button prints a page. If you would prefer seeing a print dialog box first, choose Print from the File Menu.

You can hide the Tool Bar using the View Menu.
