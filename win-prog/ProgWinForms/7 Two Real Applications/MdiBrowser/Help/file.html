The File Menu

The File menu primarily lets you open a new web page, or print a web page.

The New menu item creates a new child window with the same page as the active child window. If there is no child window active, the New item opens a new child window with your Home page.

The Open menu item lets you type in a URL of a web site, or browse your local file directory for a file to load.

The Save As command lets you save an HTML page and accompanying files to your local disk.

The Page Setup command lets you set up a page definition in preparation for printing.

The Print command displays a print dialog box that lets you print a web page.

The Print Preview command displays a preview of the page that will be printed by the Print command.

The Properties command displays information about the current web page.

The Exit command terminates the MDI Browser program.
