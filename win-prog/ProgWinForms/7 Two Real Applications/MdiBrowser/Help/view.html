The View Menu

The Tool Bar item on the View menu lets choose to display or not display the program Tool Bar.

Similarly, the Address Bar item lets you choose to display or not display the program Address Bar

The first five commands are related to navigation:

The Back command navigates to the previous web page in the currently active child window. This item will be disabled if there isn't a previous web page.

The Forward command reverses the navigation of the Back command.

The Stop command stops the execution of a lengthy download.

The Refresh command refreshes the contents of the current page in the currently active child window.

The Home command goes to your Home page, which you can set on the Favorites Menu.

The Source comand lets you view the text source of an HTML page.
