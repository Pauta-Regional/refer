Program Overview

MDI Browser is a web browser with an architecture based around the Multiple Document Interface (MDI). Each child window displays a different web page and can be independently navigated.

Aside from the use of MDI, the user interface of this program closely resembles that of Microsoft Internet Explorer. Indeed, this program uses the Internet Explorer "engine" in the form of a class named System.Windows.Forms.WebBrowser available in the .NET Framework 2.0.

This program was written by Charles Petzold for the book Programming Microsoft Windows Forms (Microsoft Press, 2005) in August 2005.
